This package shows a selected jQuery Function on api.jquery.com

How to use
==========

Just click on a jQuery Function while holding down ALT key or select a jQuery Function and press ALT+J.
A new window will be created in your default browser loading http://api.jquery.com/SELECTION.

For more information visit: https://github.com/Miw0/jQueryDocs
