This package shows the selected jQuery Function on api.jquery.com

How to use
==========

Select a jQuery Function (without braces, dot or semicolon) and press the assigned key combination (Default ALT+J).
A new window will be created in your default browser loading api.jquery.com with the corresponding documentation page for the selected function.

Change default key binding
==========================

You can change the default key combination by including this in your user key binding settings

    { "keys": ["alt+j"], "command": "jquery_docs" }
